# Ander507 Studio

Welcome to **Ander507 Studio**! This is the source code for my personal digital portfolio, showcasing a collection of modern web applications, Next.js projects, Minecraft mods, and experimental utilities.

You can visit the live site here: **[ander507.studio](https://ander507.studio)**

## 🚀 Featured Projects

- **[Ztionix](https://www.ztionix.tech/)**: A collection of experimental projects and digital utilities, featuring the Sakovajo Protocol brainrot translator.
- **[zlib.lol](https://www.zlib.lol/)**: The smart link generator. A highly secure platform to drag files, paste text, or shorten URLs instantly.
- **[NoteForge AI](/noteai)**: An AI-Powered Study Notes generator. Upload lectures, PDFs, or slides and get perfect Obsidian-ready Markdown in seconds.
- **[PacketSpy](https://modrinth.com/mod/packet-spy)**: A Minecraft mod made to debug and log packets sent/received by the client via Java WebSockets.
- **[CatzyCraft](https://modrinth.com/modpack/catzycraft)**: A fun and tech-focused multiplayer Minecraft modpack.

## 🛠️ Tech Stack

The portfolio itself is built to be lightweight, fast, and static:
- HTML5 / CSS3 / Vanilla JavaScript
- Hosted and deployed on [Vercel](https://vercel.com/)
- Vercel Analytics integrated

*(Individual projects linked on the portfolio use their own stacks, heavily featuring Next.js, React, and Java).*
